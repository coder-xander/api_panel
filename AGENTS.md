# AGENTS.md

## Project Guidelines

### Code Analysis
- When starting work on this project, always begin by analyzing the codebase using the **`codebase-memory`** MCP tool to understand the code structure, dependencies, and architecture.

### Commit Messages
- All commit messages must be written in **English**.
- Prefix each commit message with a relevant, visually appealing **emoji** to improve readability and quick scanning.

Examples:
- `✨ Add user authentication module`
- `🐛 Fix session timeout handling`
- `♻️ Refactor database connection pooling`
- `📝 Update API documentation`
- `⚡ Improve query performance with indexing`
